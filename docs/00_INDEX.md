## Operating Rules（事故防止）
- 仕様/運用を変更する場合：**先に decisions/ に ADR を追加** → その後 docs を変更
- 用語が増えた場合：**glossary/** に追加して揺れを止める
- 根拠（原文・スクショ・ログ）：**sources/** に保存
- 検証手順（再現可能な形）：**checks/** に追加
